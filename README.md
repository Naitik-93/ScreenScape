# ScreenScape
ScreenScape
Explore the World of Movies
📌 Project Overview

ScreenScape is a modern and responsive movie exploration web application that allows users to search, discover, and interact with movies using real-time data from the OMDb API. The application provides a clean and user-friendly interface for a seamless experience across devices.

🎯 Objective

The objective of this project is to demonstrate the implementation of JavaScript concepts such as API integration, array higher-order functions, and dynamic UI rendering.

🌐 API Used

OMDb API (Open Movie Database) is used to fetch movie data including title, year, type, poster, and detailed information.

🚀 Features
🔍 Search with Debouncing
Search movies by title
Optimized using debouncing

🎯 Filtering
Filter by year range
Filter by type (movie, series, episode)
Implemented using .filter()

📊 Sorting
Sort movies by year
Implemented using .sort()
❤️ Favorites
Add movies to favorites
Stored using localStorage
Rendered using .map()
🎬 Movie Details
View plot, actors, director, rating in a modal
🌙 Dark / Light Mode
Toggle theme using CSS variables
⏳ Loading Indicator
Shows loading spinner during API calls

🔄 Scroll
Loads more movies while scrolling
